Introduction

This is a demo built on ServerHub's WebSocket APIs. This help article will introduce the features and usage of this RTC demo. To catch up with ServerHub's WebSocket API, you need to installserverhub-mvcfrom npm registry (Sugessted version isv1.6.2+).

ServerHub Open Source program has provided a lot documents about this, check it out at docs.serverhub.io.

This RTC demo provide you ability to chat with anyone all over the world without providing any of your information. Your privacy is not been collected. All source code is available at GitHub - ServerHubOrg/serverhub-template#feature-chat.

Chat History

As we don't use cookie in this demo, your connection to the remote server is state-less. So everytime you refresh your browser, the same account name and avatar will be shown but all the previous conversations are cleared. You can continue talking with other online users.

If you did refresh the page, other users might have a notice that you left the chat and then returned. That's why your privacy/security counts the most!

Everything Starts from Home

You may find a input box from the home page. Simply input a nick name for yourself like Wayne, and click the paper airplane button (or tap 'Enter' on your keyboard).

Note: available nick names always starts with a-z or a '_'. Then followed by several digits, a...z, '_' or '-'. But '-' is not allowed to be the trailing character.

After successfully entered the chatroom, there is nothing to say. Enjoy!
